    • 环境准备
    • 开发
    • 调试

为了能正常开发调试手机app，需要安装

  • Node.js
  • Android Studio
  • JDK1.8
  • Android SDK

并配置相应的环境变量。

一部android手机或安卓模拟器。 手机需要开启调试模式 模拟器需要连接对应端口，已网易MuMu模拟器举例，本地端口为7555。 在打开模拟器后，命令行中执行adb connect 127.0.0.1:7555 可以通过adb devices查看已连上哪些设备。

在连上调试设备后：

  • 运行npm run run-android编译打包开发包部署到设备上
  • 运行npm run build-blue编译生产包部署到设备上 其他常用指令可以在项目根目录package.json的scripts中查看。 也可以自己根据需求在根目录build文件夹下新增打包配置文件，并添加到package.json的scripts中。
